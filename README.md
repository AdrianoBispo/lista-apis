# Lista de API's Públicas
As APIs listadas são gratuitas ou com planos free, perfeitas para projetos pessoais, MVPs, estudos ou integração em apps reais. Essas APIs públicas são ferramentas incríveis que possibilitam a integração de dados e funcionalidades de grandes plataformas em seus projetos. Elas podem transformar uma aplicação simples em algo interativo, educativo e até divertido!

**Confira algumas das APIs públicas mais populares e o que você pode construir com elas:**

## 🎬 **Entretenimento e Cultura Pop**

### 📌 **Marvel API**

- **Tipo:** REST
- **Site oficial:** [developer.marvel.com](https://developer.marvel.com/)
- **Como usar:** Criar uma conta na Marvel Developer, gerar uma public key e private key, e realizar chamadas autenticadas com timestamp, hash (MD5) e sua chave.
- **Exemplo de projeto:** App que compara superpoderes entre heróis, timeline interativa de eventos da Marvel, quiz de personagens.

### 📌 **SWAPI (Star Wars API)**

- **Tipo:** REST
- **Site oficial:** [swapi.dev](https://swapi.dev/)
- **Como usar:** Sem autenticação; basta fazer chamadas REST padrão (ex: `/people/1/`).
- **Exemplo de projeto:** Enciclopédia da saga, cards colecionáveis digitais, bot de curiosidades no Telegram/Discord.

### 📌 **Rick and Morty API**

- **Tipo:** REST
- **Site oficial:** [rickandmortyapi.com](https://rickandmortyapi.com/)
- **Como usar:** Sem autenticação, endpoints acessíveis via GET.
- **Exemplo de projeto:** Criador de cards de personagens, quiz de episódios, jogo de adivinhação.

### 📌 **Jikan (MyAnimeList API alternativa)**

- **Tipo:** REST
- **Site oficial:** [jikan.moe](https://jikan.moe/)
- **Como usar:** Acesso gratuito sem chave de API, basta consumir via endpoints como `/v4/anime`.
- **Exemplo de projeto:** Catálogo de animes assistidos, app de recomendações, comparador de personagens.

### 📌 **Studio Ghibli API**

- **Tipo:** REST
- **Site oficial:** [ghibliapi.vercel.app](https://ghibliapi.vercel.app/)
- **Como usar:** GET para endpoints como `/films`.
- **Exemplo de projeto:** Galeria dos filmes, linha do tempo de lançamentos, app para descobrir curiosidades.

### 📌 The Lord of the Rings API

- **Tipo:** REST
- **Site oficial:** [the-one-api.dev](https://the-one-api.dev/)
- **Como usar:** GET para endpoints como `/book`.
- **Exemplo de projeto:** App que exibe informações sobre livros, filmes e personagens.

### 📌 The Open Movie Database (OMDb)

- **Tipo:** REST
- **Site oficial:** [omdbapi.com](https://www.omdbapi.com/)
- **Como usar:** Criar uma conta, gerar uma public key e private key, e realizar chamadas autenticadas com timestamp, hash (MD5) e sua chave.
- **Exemplo de projeto:** Aplicativos de recomendação de filmes, catálogos de cinema, plataformas de avaliação de conteúdo.

## 🏛️ **Dados Públicos e Governamentais**

### 📌 Banco Central do Brasil API

- **Tipo:** REST
- **Site oficial:** [dadosabertos.bcb.gov.br](https://dadosabertos.bcb.gov.br/)
- **Como usar:** Acessar datasets por CSV, JSON ou endpoints de APIs específicas.
- **Exemplo de projeto:** Acesso a dados do sistema financeiro nacional, incluindo cotações de moedas, taxas de juros e informações sobre instituições financeiras.

### 📌 **Portal da Transparência**

- **Tipo:** REST
- **Site oficial:** [portaldatransparencia.gov.br](https://portaldatransparencia.gov.br/api-de-dados)
- **Como usar:**  Para fazer uso da API, deve-se realizar o cadastro de um e-mail em http://portaldatransparencia.gov.br/api-de-dados/cadastrar-email. Através do e-mail cadastrado, você receberá um token, que deverá ser usado nas suas consultas via API.
- **Exemplo de projeto:** API que disponibiliza dados sobre despesas do governo, contratos, convênios e servidores públicos que pode ser utilizado em apps de fiscalização de gastos públicos, plataformas de jornalismo de dados, aplicativos de controle social.

### 📌 **Catálogo de APIs Governamentais**

- **Tipo:** REST
- **Site oficial:** [gov.br](https://www.gov.br/conecta/catalogo/)
- **Como usar:**  Centraliza diversas APIs de diferentes órgãos do governo, abrangendo temas como CPF, CNPJ, CadÚnico, entre outros.
- **Exemplo de projeto:** Sistemas de validação de dados cadastrais, aplicativos que facilitam o acesso a serviços públicos, plataformas de integração de dados governamentais.

### 📌 **DATASUS API**

- **Tipo:** REST
- **Site oficial:** [datasus.saude.gov.br](https://datasus.saude.gov.br/cadastramento-de-ip-para-acesso-a-api-de-consumo-de-dados-no-e-sus-notifica/)
- **Como usar:**  Oferece acesso a dados do Sistema Único de Saúde (SUS), como informações sobre estabelecimentos de saúde, produção hospitalar e estatísticas vitais.
